# TravelSpots
オススメのスポットを共有するサイトです。
自分のオススメの場所をマップ上で共有して色々な人に共有できます。

# URL
画面中央のログインボタンからメールアドレスとパスワードを入力してログインできます。

# 使用技術
- php
- laravel
- PostgreSQL
- node.js
- Docker/Docker-compose
- Google Maps API
- tailwindcss

# 機能一覧
- ユーザー登録、ログイン機能(Breeze)
- googlemap表示画面
    - 位置情報検索機能(geocoder)
    - マーカー表示(marker)
    - マーカー吹き出し表示(infowindow)
